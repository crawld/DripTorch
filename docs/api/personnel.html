Burn operation personnel

class driptorch.personnel.Igniter(velocity: float, rate: float, rate_units: str = 'meters')

Bases: object

An igniter is anything with a velocity and ignition interval, like a person carrying a drip torch or a drone dispatching DAIDs.

class driptorch.personnel.IgnitionCrew(same_velocity: bool = True, same_rate: bool = True)

Bases: object

An ignition crew is a collection of igniters. Sometime you may want your igniters to all have the same velocity and/or interval. You can specifiy these constraints in the constructor.
