class driptorch.firing.Strip(burn_unit: driptorch.unit.BurnUnit, ignition_crew: driptorch.personnel.IgnitionCrew)

Bases: driptorch.firing._base.FiringBase

generate_pattern(spacing: float, depth: float, heat_depth: Optional[float] = None, side: str = 'right') → driptorch.pattern.Pattern

Generate a strip head fire ignition pattern.

Parameters

  • spacing (float) – Staggering distance in meters between igniters within a heat

  • depth (float) – Horizontal distance in meters between igniters and heats

  • heat_depth (float) – Depth in meters between igniter heats. If None, heat_depth is equal to igniter depth. Defaults to None.

  • side (str) – Side of the wind vector to start the ignition. Defaults to ‘right’. Options are ‘left’ or ‘right’.

Returns

Spatiotemporal ignition pattern
